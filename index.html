Sobre

Bem-vinda(o) ao meu Portfólio de Projetos de DataViz.

Sou Anderlane Oliveira, analista/cientista que ama de descobrir novas soluções com o uso dos dados!

Este portfólio tenho como objetivo apresentar uma amostra de designs para vizualização de dados desenvolvidos ao longo da minha jornada com dados e com o uso das principais ferramentas disponíveis no mercado.

Os dados apresentados representam amostras de fontes abertas ou dados fictícios.
